
# Alfred

## Workflows

| Workflows                              | 描述                                                      |
|----------------------------------------|-----------------------------------------------------------|
| [IP Address](#获取IP地址)              | 获取局域网IP/网外IP                                       |
| [Create New File in Finder](#新建文件) | 在当前Finder路径下新建文件                                |

### 获取IP地址

唤起`Alfred`，输入`ip`，选择要获取的IP，回车复制到系统粘贴板中

### 新建文件

在当前Finder路径下新建文件

唤起`Alfred`，输入`new 文件名`，回车
